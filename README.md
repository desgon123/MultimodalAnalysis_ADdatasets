Multimodal Integration of Transcriptomics and Proteomics in Alzheimer’s Disease

Overview

This project demonstrates a multimodal data integration approach combining transcriptomic and proteomic datasets to identify molecular signatures associated with Alzheimer’s disease.
While many studies analyze single modalities in isolation, integrating multiple omics layers can reveal more robust and biologically meaningful patterns.


Datasets

•	Transcriptomics: GSE5281 (brain tissue microarray) 

•	Proteomics: GSE29652 (brain proteomics) 

The first dataset was obtained from Gene Expression Omnibus, whereas the second dataset was obtained from a github repository (https://github.com/hazhay/Meta-analysis_AD/tree/main/Data)

Methods

1. Single-modality analysis
-	Normalization and preprocessing 
- Differential expression (transcriptomics) 
- Differential abundance (proteomics)

2. Multimodal integration
- Mapping genes ↔ proteins 
- Overlap analysis of significant features 
- Correlation analysis across modalities 
- Joint dimensionality reduction (principal component analysis (PCA))
 
3. Biological interpretation
- Pathway enrichment analysis (GO)
- Identification of Alzheimer’s-relevant pathways

Key Results

•	Identified overlapping molecular signatures across RNA and protein levels 
•	Observed moderate concordance between transcriptomic and proteomic changes 
•	Highlighted pathways related to neurodegeneration and synaptic function

Motivation

•	This project was developed to build expertise in multimodal data integration and demonstrate the ability to combine heterogeneous biological datasets.
